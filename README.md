# Conway's Game of Life

## Description

This is a simple Rust implementation of the Game of Life, a cellular automaton devised by John Horton Conway.

## Building

```bash
cargo build --release
```

## Running

```bash
cargo run
```

## Dependencies

```toml
[dependencies]
rand = "0.8.5"
termsize = "0.1.8"
```
